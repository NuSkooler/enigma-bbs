Installation Methods

Installation Methods

There are multiple ways of installing ENiGMA BBS, depending on your level of experience and desire to do things manually versus have it automated for you.

Method Operating System Compatibility Notes
Installation Script Linux, BSD, OSX Quick and easy installation under most Linux/UNIX like environments (Linux, BSD, OS X, …)
Docker Images Linux, BSD, OSX, Windows Easy upgrades, compatible with all operating systems, no dependencies to install
Manual Linux, Windows, BSD (And others; YMMV!) If you like doing things manually, or are running Windows
